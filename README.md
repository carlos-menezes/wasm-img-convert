# wasm-img-convert
> In-browser image conversion powered by Rust and WebAssembly.

![](https://i.imgur.com/xnFDoW9.gif)

## Getting Started
1. Install NodeJS and Rust
2. `npm i`
3. `npm run build:wasm`
4. `npm i ./wasm/bin`
5. `npm run start`
